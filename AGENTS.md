# Agent rules for this repo

## Formatting (required)

- If you change any C/C++ source or header files, run `./scripts/format.sh` before finishing the task.
- If you are asked to create a commit, always run `./scripts/format.sh` immediately before staging/committing.
- If formatting changes files after staging, re-stage the files and only then proceed with the commit.

## Commits

- Do not create commits unless the user explicitly asks.
